# Learning Julia through Advent of Code

This repository contains my ongoing solutions to [Advent of Code](https://adventofcode.com/) puzzles.
I first took on the challenge of completing the daily puzzles in the 2023 edition, but I will also post solutions to previous years' puzzles as I do them over time.

My goal for this challenge, apart from earning all 50 stars by Christmas, is to learn and practice Julia.
